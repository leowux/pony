Manage HUD statusline settings.

```bash
pony hud [action]
```

### Actions

- `on` - Enable HUD
- `off` - Disable HUD
- `status` - Show current HUD configuration
- (no action) - Run HUD rendering

### Examples

- `/pony:hud on` - Enable HUD
- `/pony:hud off` - Disable HUD
- `/pony:hud status` - Show configuration
